# Audit Qualite Complet

Audite tous les livrables marketing du projet en cours.

## Instructions

1. **Inventaire** : Liste tous les fichiers dans `output/` (landing-pages, emails, ads, content, lead-magnet)

2. **Evaluation** : Pour chaque livrable, utilise l'agent **reviewer** pour l'evaluer selon la grille appropriee

3. **Rapport** : Compile les resultats dans `output/quality-audit.md` :

```

|---------|------|-------|---------|

## Classement par priorite de correction

1. [Fichier le plus urgent] : [Probleme principal] -> [Action]
2. [Fichier suivant] : [Probleme] -> [Action]

## Top 3 ameliorations a impact maximal

1. [Amelioration qui impacte le plus la conversion]
2. [Amelioration suivante]
3. [Amelioration suivante]

## Score global du funnel : XX/100
```

4. **Recommandation** : Suggere les corrections a apporter en priorite et les skills a utiliser pour chaque correction
