# Brand Voice -- Formation/Coaching

## Objectif

Capturer le style d'ecriture unique du createur/coach et l'appliquer
a tous les contenus marketing pour garantir la coherence.

## Processus

### Etape 1 : Analyser le style existant

Lire `docs/brand-guidelines.md`. S'il est vide, demander :
- 3-5 exemples de contenus ecrits par le createur (posts, emails, pages)
- Ou une description du ton desire

Extraire :
- **Registre** : Formel / Semi-formel / Decontracte / Familier
- **Personnalite** : Expert autoritaire / Ami bienveillant / Mentor provocateur / Guide pragmatique
- **Vocabulaire** : Mots favoris, expressions recurrentes
- **Structure** : Phrases courtes vs longues, listes, storytelling
- **Tabous** : Mots ou expressions a ne jamais utiliser

### Etape 2 : Creer le guide de voix

Remplir `docs/brand-guidelines.md` avec ce format :

```
## Ton en 3 mots : [Mot 1], [Mot 2], [Mot 3]

## Qui parle ?
[Description du personnage, 2-3 phrases]

## Regles d'ecriture
- [Regle 1 : ex. "Tutoiement systematique"]
- [Regle 2 : ex. "Phrases de max 15 mots"]
- [Regle 3 : ex. "Toujours un exemple apres une affirmation"]

## Expressions favorites
- [Expression 1]
- [Expression 2]

## Mots/expressions interdits
- [Interdit 1 + pourquoi]
- [Interdit 2 + pourquoi]

## Test du ton
Phrase generique : "Apprenez le marketing digital"
Notre version : "[Reformulation dans le ton]"
```

### Etape 3 : Appliquer le ton (quand utilise sur un contenu existant)

1. Lire le contenu a adapter
2. Identifier les passages hors-ton
3. Proposer des reformulations alignees
4. Verifier la coherence globale

## Criteres de qualite

- Un lecteur reconnaitrait l'auteur sans voir son nom
- Le ton est coherent sur tous les supports
- Le ton est SPECIFIQUE (pas "professionnel et amical")
- Les exemples sont concrets et clairs
- Le guide tient en 1 page max

## Erreurs courantes a eviter

- Ton generique qui decrit tout le monde
- Ton copie d'un autre createur
- Guide trop long que personne ne relit
- Ne pas mettre a jour quand le style evolue
